# types.ContentImage

``` python
types.ContentImage()
```

Base class for image content.

This class is not meant to be used directly. Instead, use [`content_image_url`](https://posit-dev.github.io/chatlas/reference/content_image_url.html#chatlas.content_image_url), [`content_image_file`](https://posit-dev.github.io/chatlas/reference/content_image_file.html#chatlas.content_image_file), or [`content_image_plot`](https://posit-dev.github.io/chatlas/reference/content_image_plot.html#chatlas.content_image_plot).
