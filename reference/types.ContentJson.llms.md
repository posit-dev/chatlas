# types.ContentJson

``` python
types.ContentJson()
```

JSON content

This content type primarily exists to signal structured data extraction (i.e., data extracted via [`Chat`](https://posit-dev.github.io/chatlas/reference/Chat.html#chatlas.Chat)’s `.chat_structured()` method)

## Parameters

| Name  | Type | Description             | Default    |
|-------|------|-------------------------|------------|
| value |      | The JSON data extracted | *required* |
