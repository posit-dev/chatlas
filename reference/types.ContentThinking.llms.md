# types.ContentThinking

``` python
types.ContentThinking()
```

Thinking/reasoning content

Captures the model’s internal reasoning process.

## Parameters

| Name | Type | Description | Default |
|----|----|----|----|
| thinking |  | The thinking/reasoning text from the model. | *required* |
| extra |  | Additional metadata associated with the thinking content (e.g., encrypted content, status information). | *required* |
